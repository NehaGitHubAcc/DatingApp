# Full-Stack Dating Application 💑

A comprehensive full-stack web application built to demonstrate enterprise-grade development patterns using .NET Core and Angular.

## 🛠️ Tech Stack
* **Backend:** .NET Core Web API, Entity Framework Core
* **Frontend:** Angular, TypeScript, TailwindCSS , DaisyUI
* **Database:** SQL Server 
* **Security:** JWT Authentication & Password Hashing

## 🚀 Key Features
* Secure User Registration & Login (JWT-token based)
* Complete User Profile management with photo upload integration
* Real-time filtering, sorting, and pagination for user searches
* User messaging and liking system
